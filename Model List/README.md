# Model List

Model List主要列举了常用大模型的基础信息以及对应API的使用方法，比如**ChatGLM**、**LLaMA**等，也从**通用领域**和**垂直领域**进行了划分，具体如下所示：

- **通用领域**：文本大模型、文本图/视频、文生音乐、代码生成以及多模态等
- **垂直领域**：法律大模型、医疗大模型、金融大模型、环境大模型、网络安全大模型、教育大模型以及交通大模型等
