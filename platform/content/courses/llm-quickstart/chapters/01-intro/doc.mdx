# 什么是大语言模型

## 定义

大语言模型（Large Language Model, LLM）是一种基于深度学习的自然语言处理模型，通过在海量文本数据上进行预训练，获得了理解和生成人类语言的能力。

## 核心特点

| 特点 | 说明 |
|------|------|
| 规模大 | 参数量通常在数十亿到数千亿之间 |
| 预训练 | 在大规模无标注文本上自监督学习 |
| 泛化强 | 通过提示词即可适应多种任务 |
| 上下文学习 | 能从示例中学习新任务 |

## 发展历程

1. **GPT-1 (2018)** — 1.17亿参数，证明生成式预训练的有效性
2. **BERT (2018)** — 双向编码器，开启预训练时代
3. **GPT-3 (2020)** — 1750亿参数，展现惊人的涌现能力
4. **ChatGPT (2022)** — 引入 RLHF，对话能力质变
5. **GPT-4 (2023)** — 多模态、推理能力大幅提升

## 核心能力

- **文本生成**：写作、翻译、摘要
- **知识问答**：覆盖科学、历史、技术等领域
- **代码生成**：多种编程语言的编写和调试
- **逻辑推理**：数学、逻辑谜题求解
- **创意创作**：故事、诗歌、营销文案
