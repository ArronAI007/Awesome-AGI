# 提示工程基础

## 什么是提示工程

提示工程（Prompt Engineering）是指设计和优化输入提示（Prompt），以引导大语言模型产生更准确、更有用的输出的技术。

## 基础技巧

### 1. 明确指令

给模型清晰、具体的指令，避免模糊表达。

```
❌ 差：写一段关于猫的文字
✅ 好：请以科普风格写一段 200 字的介绍，说明家猫（Felis catus）的驯化历史和它们在人类社会的角色
```

### 2. 提供上下文

在提问前提供必要的背景信息。

```
你是一位有 10 年经验的 Python 开发工程师。
我正在学习异步编程，请用简单的例子解释 async/await 的工作原理。
```

### 3. 使用角色设定

让模型扮演特定角色，可以获得更专业的回答。

```
请你扮演一位高中物理老师，用生动的类比向学生解释量子纠缠的概念。
```

### 4. 分解复杂任务

将复杂问题拆分成多个步骤。

```
请按以下步骤帮我分析这段代码：
1. 先解释代码的整体功能
2. 指出潜在的性能问题
3. 给出优化建议
4. 提供优化后的代码
```

## 进阶框架

- **CoT (Chain-of-Thought)**：要求模型逐步推理
- **Few-shot**：提供几个示例后再提问
- **RAG**：结合外部知识库回答
