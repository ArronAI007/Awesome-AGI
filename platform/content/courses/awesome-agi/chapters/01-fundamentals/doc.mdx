# 大模型基础理论

## 目录

本章将带你系统了解大语言模型（LLM）的核心知识体系：

1. **微调（Fine-tuning）** — 掌握 LoRA、QLoRA、DoRA 等参数高效微调方法
2. **提示工程（Prompt Engineering）** — CoT、ToT、GoT 等高级提示框架
3. **模型评估** — 全面理解 LLM 评测指标与基准测试
4. **安全与对齐** — RLHF、幻觉检测、长上下文处理

## 微调方法概览

| 方法 | 原理 | 适用场景 |
|------|------|---------|
| LoRA | 低秩适配，冻结原权重 | 通用微调 |
| QLoRA | 4-bit 量化 + LoRA | 显存受限环境 |
| DoRA | 权重分解低秩适配 | 更高精度需求 |
| Prefix Tuning | 训练前缀嵌入 | 多任务场景 |
| Adapter | 插入小型适配层 | 持续学习 |

## 提示工程框架

- **CoT (Chain-of-Thought)**：逐步推理，让模型展示思考过程
- **ToT (Tree-of-Thoughts)**：多路径探索，像决策树一样评估不同思路
- **GoT (Graph-of-Thoughts)**：图结构推理，支持更复杂的思维聚合
